# Resumo

Desenvolvi o site institucional da Babitonga Padel, um clube de padel em São Francisco do Sul - SC. O projeto é uma landing page estática (HTML + CSS + JS vanilla, zero dependências) focada em conversão via WhatsApp.
O site apresenta o clube, diferenciais (3 quadras profissionais, bar, área gourmet), eventos e depoimentos de atletas. Todo o design system foi criado com CSS custom properties, mantendo consistência visual em toda a experiência.
